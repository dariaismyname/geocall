## General info
This is a test project where you can chat and have a video call with your friends. 

## Technologies
Project is created with:
* HTML
* JavaScript
* Tailwind
* React
* Redux Toolkit
* Peer.js
* Socket.io

## Setup
To run this project: 

### Server

```
$ cd server
$ npm install
$ npm start
```
### Client
```
$ cd client
$ npm install
$ npm start
```
